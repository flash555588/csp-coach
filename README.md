# CSP Coach — 绘画教练层原型

面向初学者的 Clip Studio Paint 跨平台学习与辅助创作系统。

基于研究报告中的"绘画教练层"理念，本原型把报告里的方法论转化为可交互的 Web 应用：不是替代用户画画，而是在每一步给出"下一步该做什么"的低摩擦指引。

---

## 功能总览

### 仪表盘
- **周次切换器**：1–12 周快速跳转，自动更新本周主题和任务
- **今日推荐**：基于当前周次智能推荐今日练习，附推荐理由
- **练习计时器**：15/30/45/60 分钟四档预设，倒计时完成后自动记录到本周进度
- **每日打卡**：月历视图显示打卡记录，连续天数统计，选择练习类型后一键打卡
- **里程碑进度**：5 个达标标准，点击即可标记完成
- **本周任务**：当前周的任务清单

### 学习路径
- **4 周速成 / 12 周系统**两套课表一键切换
- 每周卡片含阶段标签、任务、产出、达标标准
- 点击展开/收起详情

### 工作流引导
- **5 条核心工作流**：临摹起稿、配色上色、线条训练、元素构图、人物动态
- **步骤完成标记**：点击步骤编号圆圈即可标记完成，已完成步骤自动划线、降低不透明度
- **工作流进度条**：顶部显示当前工作流完成百分比
- **导出前复盘**：工作流全部完成后可打开复盘弹窗，按构图/主次/值关系/手部四项逐一检查
- 分步操作说明 + 推荐工具 + 验证标准

### 配色方案
- **6 组精选手工配色**：暖棕稳妥、蓝灰稳妥、薄荷清爽、樱花清爽、紫橙戏剧、红黑戏剧
- **随机生成**：基于 HSL 色彩空间按三种模式随机生成
- **三档情绪分类**：稳妥（单色系）、清爽（邻近色）、戏剧化（互补色）
- **点击复制**：色条或色块点击复制 HEX 到剪贴板，支持复制整组配色
- 每组含主色/辅色/点缀/阴影/高光五色

### 参考拆解
- **本地上传**：支持拖拽或点击上传参考图
- **三种观察模式**：大形（模糊降低细节）、轮廓（高对比提取边缘）、明暗（去色看值关系）
- **观察要点**：每种模式配 4 条具体观察提示
- **使用说明**：三种模式的作用对比

### 资源库
- 15 个配套工具，按 9 个分类筛选
- 每个工具含平台、费用、用途、外部链接、标签

### 进度追踪
- **本周概览**：四项指标（临摹/线条/配色/动态）带进度条和目标对比
- **每周计数器**：12 周全部展开，每项指标带 +/- 按钮
- **周完成标记**：一键标记整周完成

### 提示词模板
- 8 个低认知负担的 AI 提示模板
- 点击卡片一键复制到剪贴板
- 每条附使用技巧

## 技术栈

- **Vue 3** + **TypeScript** + **Vite**
- **Vue Router**（Hash 模式，适合静态部署）
- **Pinia**（状态管理，localStorage 持久化）
- **UnoCSS**（原子化 CSS，图标通过 `@iconify-json/lucide`）
- **Lucide Icons**

## 数据持久化

所有进度数据自动保存到 `localStorage`，刷新页面不丢失：
- 里程碑完成状态
- 每周练习计数
- 当前周次与当前任务
- 工作流步骤完成状态
- 每日打卡记录与连续天数
- 深色模式与侧边栏折叠状态

## 设计决策

### 深色模式优先
默认启用深色模式，减少绘画用户长时间看屏幕的视觉疲劳。

### 低认知负担
- 每个工作流最多 5 个步骤
- 提示词模板限定数量（3个配色/4个缩略图/4项复盘）
- 资源卡片只显示最关键信息
- 计时器只有 4 个预设时长，不做自定义输入
- 复盘清单只有 4 项，每项一句话
- 配色方案只给 3 档情绪选择，不开放全参数调节

### 可量化
进度追踪提供具体数字指标和目标对比：
- 临摹/草图：周目标 10
- 线条页：周目标 7
- 配色方案：周目标 4
- 动态速写：周目标 30

### 状态持久化
所有交互状态自动保存，用户关闭浏览器后再打开，进度不丢失。

## 部署

构建产物为纯静态文件，已配置 `base: './'`，可部署到任意静态托管：

```bash
npm run build
# dist/ 目录即为部署包
```

兼容 GitHub Pages、Vercel、Cloudflare Pages、Netlify 等。
